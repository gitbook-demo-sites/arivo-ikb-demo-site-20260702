# Authorized users and account updates

| Policy area | Imported subtopic |
| --- | --- |
| Authorized Users | Spousal authorization |
| Updating Account Information | Cease and desist flags, account detail changes |

{% hint style="info" icon="id-card" %}
Authorization guidance should be easy to find from search because it often determines whether a servicing action can continue.
{% endhint %}

## Account update workflow

{% stepper %}
{% step %}
### Verify authority

Confirm the person requesting the update is authorized.
{% endstep %}

{% step %}
### Check restrictions

Review cease and desist, communication preferences, and required documentation.
{% endstep %}

{% step %}
### Update the system

Make the change in the approved source system and record the policy reference.
{% endstep %}
{% endstepper %}
