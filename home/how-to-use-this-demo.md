# How to use this demo

This site is structured as a multi-section internal knowledge base:

| Section | Purpose |
| --- | --- |
| Home | Company context, search-first entry point, and demo navigation. |
| Policies | Imported policy content, split into GitBook pages instead of a hosted file. |
| Changelog | Dummy update history for review cycles and policy releases. |

## Walkthrough path

{% stepper %}
{% step %}
### Start on the homepage

Point out the branded Arivo entry point, prominent search, and card-based paths into the content.
{% endstep %}

{% step %}

Show that the Account Servicing policy is imported as structured GitBook content with metadata, callouts, and procedure pages.
{% endstep %}

{% step %}

Show how policy owners could announce updates and keep teams aligned on changes.
