# Payment waterfall

Payment waterfall guidance explains the order in which balances, fees, charges, and principal are reduced.

## Balance reduction workflow

{% stepper %}
{% step %}
### Confirm account status

Validate whether the account is active, paid, non-accrual, or in another restricted status before applying the waterfall.
{% endstep %}

{% step %}
### Apply active balance reduction

Apply the approved order of reductions for active balances and verify whether any pre-payment logic applies.
{% endstep %}

{% step %}
### Recover fees and charges

Use the relevant fee and charge recovery guidance before reducing principal.
{% endstep %}

{% step %}
### Reduce principal

Apply the principal balance reduction rules using the cost recovery method where required.
{% endstep %}
{% endstepper %}
